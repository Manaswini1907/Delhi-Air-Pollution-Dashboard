# Delhi Air Pollution Dashboard

## 📌 Overview
This project is an interactive Tableau dashboard that analyzes Delhi's air pollution data and visualizes pollutant trends.

## 🚀 Features
- Interactive Tableau Dashboard
- Air Quality Analysis
- Pollutant Comparison
- Trend Visualization
- Clean and Easy-to-understand Charts

## 🛠️ Tools Used
- Tableau Desktop
- Git
- GitHub
